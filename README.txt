To run the solution:
1. Ensure you have Spark installed and properly configured.
2. Place the dataFIFA.csv file in the same directory as the Jupyter Notebook.
3. Open the Jupyter Notebook and run the cells sequentially.
4. The notebook performs feature engineering, clustering, PCA, and visualization of clusters.

Special Instructions:
- Ensure the dataFIFA.csv file is formatted correctly and contains all necessary columns.
- The notebook assumes all currency values are in EUR and handles conversion of "M" for millions and "K" for thousands.
